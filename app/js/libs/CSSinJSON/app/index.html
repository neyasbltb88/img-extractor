Все стили на этой странице(кроме подсветки кода) вставлены с помощью CSSinJSON

Инициализация плагина

const CssInJson = new CSSinJSON({
    // Стили в виде объекта или строки JSON
    style: {object | string},
    
    // Селектор(ы) корневого элемента, если передан(ы) - включаются изолированные стили внутри него(них)
    scopedElem: {string | array} // Необязательный параметр
});

Пример изоляции стилей: 


    
.test внутри .scoped_class





    
.test внутри .scoped_class2




.test рядом со .scoped_class

Блоки выше имеют следующую структуру:

<div class="scoped_class">
    <div class="test">.test внутри .scoped_class</div>
</div>

<div class="scoped_class2">
    <div class="test">.test внутри .scoped_class2</div>
</div>

<div class="test">.test рядом со .scoped_class</div>

Как можно заметить, все 4 блока имеют класс .test, но красные только первые 3 из них. При этом, описывая стили, мы указываем красный цвет для всех классов .test. Это достигается благодаря тому, что при инициализации плагина мы передаем второй, не обязательный параметр, где перечисляем селекторы тех блоков, внутри которых хотим применять передаваемые стили. На всю остальную страницу они влиять не будут. Это позволяет решить многие проблемы с пересечением имен селекторов.

Стили для них инициализированы следующим образом:

const CssInJson2 = new CSSinJSON({
    style: {
        '.scoped_class, .scoped_class2': {
            paddingLeft: '30px'
        },
        '.test': {
            color: 'red'
        },
        '.test:before, .test:after': {
            content: '" *** "',
        }
    },
    scopedElem: ['.scoped_class', '.scoped_class2']
});
